# CLAUDE.md — project rules for Claude Code

## SEO docs — single source of truth

**`docs/marketing/seo/TODO.md` is the only place for SEO tasks, action items, and open work.**

The full rule is defined in [`docs/marketing/seo/README.md`](./docs/marketing/seo/README.md) and applies to all agents and contributors. Short version:

1. **Never create a new `TODO*.md`** anywhere under `docs/marketing/seo/`.
2. **Never create a new `ACTION-PLAN*.md`** (or any file whose primary purpose is a task list).
3. Open tasks from research → add to `docs/marketing/seo/TODO.md`, not a new file.
4. Research output (reports, battle cards, etc.) may be saved in the appropriate subfolder; only the *task list* goes in `TODO.md`.
5. Duplicate task lists found in subfolders → consolidate into `TODO.md` and delete the duplicate.
