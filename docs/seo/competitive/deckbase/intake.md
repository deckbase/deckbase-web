# Deckbase — Intake

## Product
**Deckbase** is an AI-powered flashcard mobile app (Flutter, iOS + Android) that helps learners turn notes, PDFs, and articles into AI-generated flashcards, reviewed with FSRS spaced repetition.

## Problem
Learners spend too much time re-reading source material (textbooks, lecture notes, documents) instead of testing themselves. Deckbase gives a faster path from raw material to long-term memory retention.

## Target users
- Students preparing for exams
- Medical/health learners with dense reading
- Language learners building vocabulary
- Professionals learning technical topics

## Key differentiators
- AI card generation from source material (notes, PDFs, articles)
- FSRS spaced repetition (same algorithm family as Anki)
- Rich template system (quiz blocks, audio, images, OCR)
- Anki import (.apkg)
- Offline-first with Firebase cloud sync
- Freemium with in-app purchases (RevenueCat)

## Market / category
Mobile flashcard apps / AI study tools / spaced repetition apps

## Known competitors (from ASO keyword targeting)
- **Anki** — open-source SRS flashcard tool (import source + alternative keyword)
- **Quizlet** — dominant flashcard app (keyword: "quizlet alternative")

## Tech stack
Flutter, Firebase (Auth/Firestore/Storage), Drift/SQLite, Riverpod, Anthropic Claude API (AI generation), RevenueCat (IAP)
