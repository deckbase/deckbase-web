## 3) Visual assets checklist

Prepare before launch:

- Product Hunt logo (240x240)
- Gallery images (at least 3, ideally 5-7)
- 30-60 sec demo video (optional but strongly recommended)
- Maker profile photo and clean background header

Recommended gallery sequence:

1. Hero value prop ("Turn content into flashcards instantly")
2. Input sources (PDF/article/notes)
3. AI generation flow
4. Study mode / spaced repetition
5. Pricing / plans snapshot
6. Social proof or user quote (if available)

Image copy tips:

- One message per frame
- Large text, high contrast, mobile readable
- Show outcome (memory retention, time saved), not only features

## 4) Launch timeline (T-14 to T+2)

### T-14 to T-7 (prep week)

- Finalize listing copy and visuals
- Create a "Product Hunt launch" email draft for supporters
- Make a supporters list (friends, users, communities, advisors)
- Prepare FAQ answers for likely questions
- Ensure signup/onboarding is stable and fast

### T-3 to T-1

- Dry run all links (PH, website, onboarding, pricing)
- Schedule social posts (X, LinkedIn, relevant communities)
- Confirm team response shifts so comments are answered quickly
- Prepare "thank you" DM template for supporters

### Launch day (00:01 PT preferred)

- Publish on Product Hunt at start of day
- Post maker comment in first 5 minutes
- Share launch link to supporter list (no vote begging; ask for feedback)
- Reply to every comment quickly, with specifics
- Post 2-3 updates throughout the day (progress + insights)

### T+1 to T+2 (post-launch)

- Send thank-you update with key learnings
- Collect all feedback into one backlog
- Ship one visible improvement fast and share update
- Repurpose PH content into case-study/social posts
